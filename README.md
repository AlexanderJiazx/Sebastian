## Instructions

### Installation

Install the necessary resources:

```bash
pip install google.generativeai
pip install Pillow
```

### Running Sebastian

Run Sebastian with Python:

```bash
python3 Sebastian_run.py
```

### Usage

Input your instructions and wait for a response. Sebastian will take action automatically.
